# Черновик Data safety для Play Console

Финальные ответы подтверждает владелец аккаунта разработчика перед публикацией.

## Общие ответы

- Приложение собирает или передаёт данные пользователя: **да**.
- Данные передаются третьим сторонам для рекламы или продажи: **нет**.
- Передача шифруется: **да, HTTPS**.
- Пользователь может запросить удаление: **да, внутри приложения и через сайт**.
- Реклама: **нет**.

Supabase и Vercel используются как сервисные поставщики, действующие для работы B.Nook. Open Library и Internet Archive получают только запросы, необходимые для запущенного пользователем поиска или загрузки доступного текста.

## Категории данных

| Категория Play Console | Что используется | Обязательность | Цель |
|---|---|---|---|
| Personal info → Email address | email аккаунта | обязательно для аккаунта | управление аккаунтом, безопасность |
| Personal info → Name | имя/ник профиля | имя обязательно, остальное опционально | профиль, функции сообщества |
| Photos and videos → Photos | аватар, баннер, изображения в публикациях | опционально | персонализация, пользовательский контент |
| App activity → User-generated content | публикации, комментарии, темы, ответы, цитаты | опционально | функции приложения, сообщество |
| App activity → Other actions | лайки, блокировки, жалобы, прогресс чтения | опционально | функции, безопасность, модерация |
| App info and performance → Diagnostics | технические журналы инфраструктуры | автоматически | безопасность, исправление ошибок |
| User IDs | внутренний идентификатор Supabase | автоматически | аккаунт, синхронизация, безопасность |

## Удаление и хранение

Удаление аккаунта очищает активный профиль, библиотеку, заметки и связанный пользовательский контент. Ограниченное время могут сохраняться недоступные пользователям резервные копии и технические журналы поставщиков инфраструктуры согласно их циклам хранения.
