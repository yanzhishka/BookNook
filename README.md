# B.Nook | Divine Social Reading Sanctuary

**B.Nook** — это не просто читалка или социальная сеть. Это высокотехнологичное цифровое убежище, спроектированное для тех, кто воспринимает чтение как сакральный ритуал. Платформа объединяет в себе мощный движок для чтения, инструменты коллективного разума и передовые нейросетевые технологии для глубокого погружения в мир литературы.

---

## 🏛 Основная Концепция

Проект базируется на философии "Divine Reading" (Божественное чтение). Мы создали интерфейс, который минимизирует визуальный шум и максимизирует эстетическое удовольствие. Стек технологий подобран для обеспечения мгновенного отклика и плавности, сравнимой с перелистыванием страниц дорогого коллекционного издания.

---

## 🌟 Ключевые Функции

### 1. Интеллектуальная Библиотека (The Vault)
*   **Универсальный Парсинг:** Поддержка форматов `EPUB`, `PDF` и `TXT` с автоматическим извлечением текста и метаданных.
*   **Глобальный Поиск:** Интеграция с Google Books API и Open Library для мгновенного добавления книг в коллекцию.
*   **Система Прогресса:** Точное отслеживание прочитанных страниц и автоматический расчет времени до завершения книги.

### 2. Иммерсивный Ридер (The Sanctuary)
*   **Дзен-режим:** Отключение всех отвлекающих факторов, активация эмбиент-звуков (дождь, лес) и таймера концентрации.
*   **Живые Аннотации:** Возможность выделять текст ("Liquid Selection") и мгновенно превращать его в заметку с выбором цветового кода.
*   **Типографика:** Кастомные настройки шрифтов (Serif/Sans/Mono), кегля и межстрочного интервала для снижения нагрузки на зрение.
*   **Адаптивные Темы:** Четыре пресета (Бумага, Сепия, Ночь, Солярис) для любого времени суток.

### 3. The Grid (Коллективный Разум)
*   **Анонимная Борда:** Пространство для создания тредов и глубоких обсуждений без лишнего официоза.
*   **Тепловая Карта Активности:** Треды меняют свечение и границы в зависимости от интенсивности обсуждения (Hot/Flame/Godlike).
*   **Мультимедийность:** Поддержка изображений в комментариях и всплывающая лента ответов для сохранения контекста основного поста.

### 4. Литературный Оракул & Студия
*   **Oracle (LLM Integration):** Система рекомендаций на базе Llama 3.3, которая понимает "вайб" и атмосферу запроса, а не просто жанры.
*   **Creative Studio:** Генерация визуальных образов из прочитанных сцен с помощью Gemini 3 Pro Vision. Редактируйте обложки и создавайте иллюстрации прямо в приложении.

### 5. Социальная Экосистема
*   **Community Feed:** Глобальная лента активностей, где можно делиться своими цитатами, прогрессом и мыслями.
*   **Система XP и Уровней:** Геймификация чтения. Получайте опыт за каждую прочитанную страницу, созданную заметку или комментарий.
*   **Real-time Messages:** Полноценный мессенджер на базе Supabase Realtime для личных обсуждений.

## 🛠 Технический Стек

*   **Frontend:** React 19, TypeScript.
*   **Styling:** Tailwind CSS с кастомной системой Glassmorphism и Aurora-анимациями.
*   **Backend:** Supabase (Auth, PostgreSQL, Storage, Realtime).
*   **AI Engine:** 
    *   Google Gemini API (Vision & Creative Studio).
    *   Groq API / Llama 3.3 (The Oracle).
*   **PDF/EPUB Engine:** PDF.js & JSZip.

---

## 🎨 Дизайн и UI/UX

Интерфейс выполнен в стиле **Neo-Modernism**:
*   **Aurora Backgrounds:** Живые градиенты, которые мягко меняются в зависимости от времени суток и выбранной темы.
*   **Liquid Cursor:** Кастомный курсор с эффектом ряби при нажатии, подчеркивающий "текучесть" интерфейса.
*   **Micro-interactions:** Каждая кнопка и карточка имеет физически корректный отклик (hover-lift, scale-bounce).

## 🚀 Установка и Развертывание

1.  **Клонирование репозитория:**
    ```bash
    git clone https://github.com/your-repo/bnook.git
    ```
2.  **Установка зависимостей:**
    ```bash
    npm install
    ```
3.  **Настройка окружения:**
    Создайте файл `.env` и добавьте ключи:
    *   `VITE_SUPABASE_URL`
    *   `VITE_SUPABASE_ANON_KEY`
    *   `API_KEY` (для Gemini)
    *   `GROQ_API_KEY` (для Оракула)
4.  **Запуск:**
    ```bash
    npm run dev
    ```
